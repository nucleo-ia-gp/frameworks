# EAA — Engenharia de Agentes Autonomos

**Framework for Autonomous Agent Engineering in Project Management**

> Track B — CC-BY-SA 4.0 (documentation) | MIT (code)

## Overview

The EAA (Engenharia de Agentes Autonomos) framework provides a structured methodology for designing, implementing, and governing autonomous AI agents within project management contexts.

**Developed by:** Tribo 2 — Agentes Autonomos
**Lead author:** Debora Moura (PMI-CE)
**Affiliation:** Nucleo de Estudos e Pesquisa em IA & GP — PMI Brazil

## Background Research

The EAA framework builds on comparative analysis of existing agent frameworks:

| Framework | Focus | Status |
|-----------|-------|--------|
| Autono | ReAct-based robust autonomous agents | Review |
| Gaia | General AI assistants benchmark | Review |
| MASAI | Multi-agent software engineering | Review |
| LLM-Agent-UMF | Unified modeling for LLM agents | Todo |
| ModelScope-Agent | Open-source LLM-based agents | Todo |
